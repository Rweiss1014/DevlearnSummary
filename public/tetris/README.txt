STORYLINE WEB OBJECT - READY TO USE
====================================

IMPORT INTO STORYLINE:
1. In Storyline: Insert > Web Object > From folder
2. Select THIS FOLDER (dist-storyline)
3. Set dimensions: 1280 x 720
4. Click Insert

TO CHANGE QUESTIONS:
1. Open questions_tetris.csv in this folder
2. Edit your questions
3. Save the file
4. Re-import this folder to Storyline (or just publish)

That's it! No rebuilding needed!

CSV FORMAT:
question,option1,option2,option3,option4,correctAnswer
"Your question?","Choice 1","Choice 2","Choice 3","Choice 4",1

Note: correctAnswer uses 1-4 (1 = first option)

FILES IN THIS FOLDER:
- index.html = The web object (don't edit)
- questions_tetris.csv = Your questions (edit this!)
- README.txt = This file
